margin - border 기준 바깥 여백

	box-model에서 테두리를 기준으로 
	
바깥에 양사방 적용되는 여백을 margin.
안쪽에 양사방 적용되는 여백을 padding이라 한다.;

margin의 방향별 지정 속성

  • margin-top: ;
  • margin-right: ;
  • margin-bottom: ;
  • margin-left: ;

margin의 속성 값의 갯수별 지정법

  • margin:n ; 양사방 동일하게 적용
  • margin:n n ; 상하, 좌우가 서로 동일하게 적용
  • margin:n n n ; 상, 좌우가 서로 동일하게, 하 순서로 적용
  • margin:n n n n; 상,우,하,좌 순서로 적용

margin을 이용한 center정렬법

margin-left: auto;
margin-right: auto;
or
margin: n auto;
