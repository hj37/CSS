padding - border 기준 바깥 여백

	box-model에서 테두리를 기준으로 
	
바깥에 양사방 적용되는 여백을 margin.
안쪽에 양사방 적용되는 여백을 padding이라 한다.;

padding의 방향별 지정 속성

  • padding-top: ;
  • padding-right: ;
  • padding-bottom: ;
  • padding-left: ;
green

padding의 속성 값의 갯수별 지정법

  • padding:n ; 양사방 동일하게 적용
  • padding:n n ; 상하, 좌우가 서로 동일하게 적용
  • padding:n n n ; 상, 좌우가 서로 동일하게, 하 순서로 적용
  • padding:n n n n; 상,우,하,좌 순서로 적용
