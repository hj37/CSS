transition

  • transition: ; 함축형 표현(속성값만 나열)
  • transition의 세부 속성들
    1. transition-property: ; 적용할 속성
    2. transition-duration: ; 전체 시간. 필수 (단위는 s나 ms)
    3. transition-delay : ; 지연 시간
    4. transition-timing-function : ; 가속감속 등의 효과
      1. ease 기본값
      2. ease-in 시작을 느리게
      3. ease-out 마지막을 느리게
      4. ease-in-out 시작과 마지막을 느리게
      5. linear 등속
      6. cubic-bezier(n,n,n,n) 베지에곡선을 이용 만들기
