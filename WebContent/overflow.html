overflow 영역 밖의 내용 제어

  • overflow: visible: 기본값.
    (영역과 상관없이 컨텐츠가 다 보임)
  • overflow: hidden: 영역 밖의 컨텐츠 안보임
  • overflow: scroll: 스크룰바가 항상 생성
  • overflow: auto: 영역보다 컨텐츠가 클 경우에만 해당 부분에 스크롤바 생성
