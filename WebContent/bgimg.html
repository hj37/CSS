background: ; 배경

  • 배경 반복
  • 배경 위치
  • 배경 크기
  • 배경 스크롤
  1. background-color : ; 배경 색상
    • 색상명
    • rgb(n,n,n) - 빛의 삼원색 red,green,blue(n은 0~ 255사이의 값)
    • hsl(n,n,n) - 색상 명도 채도의 순서
    • #nnnnnn - 16진수로 표현한 방식(3자리 가능)
  2. background-image:url("이미지경로"); 배경 이미지
    • background-repeat:repeat/repeat-x/repeat-y/no-repeat; 반복 제어
    • background-size: ; 크기 지정
    • background-position: ; 위치 지정
    • background-attachment: ; 스크롤 여부 지정
    * [ 배경 이미지 2개 적용하기 ]
    background-image:url("이미지1경로"), url("이미지2경로");
    background-size: 1속성값, 2속성값 :
    ...

background-repeat:repeat/repeat-x/repeat/-y/no-repeat

background-repeat:repeat
배경 이미지 양사방 반복(기본값)

background-repeat:no-repeat
배경 이미지 반복 안 함

background-repeat:repeat-x
배경 이미지 가로로만 반복

background-repeat:repeat-y
배경 이미지 세로로만 반복

background-position:수평 수직;

수평과 수직 방향값을 조합하여 지정하거나(1/c/r, t/c/b) 
여러 단위를 활용하여 직접 값을 지정할 수 있다. (npx/n%/nem/...)

background-position:right center;

background-position:center center;

background-position:50% 50%;

background-position:75px 40px;

background-size:contain/cover/n n/; 

  contain이나  cover로 정해진 값을 쓰거나 
   여러 단위를 활용하여 직접 값을 지정할 수 있다.
  (가로 세로 | npx npx/n% n%/nem nem/...)
  
  * cover = 이미지 하나로 전체를 다 덮기 
           (이미지가 잘려나갈 수 있음)
    contain = 영역에 이미지 하나를 온전히 다 보이게 
           (반복값에 따라 이미지가 반복되거나 영역에 빈공간이 생길 수 있음)

background-size:contain;

background-size:conver;

background-size:100% 100%;

background-size:contain/cover/n n/;
