CSS basic

선택자 {
  속성:속성값;
  속성:속성값;
    ...
   }

CSS 적용방법 3가지

  • External CSS - CSS소스가 HTML문서와 따로 독립되어 연결된 형태 여러 웹문서에 일관된 디자인 적용, 유지보수 용이.
  • Internal CSS - CSS소스가 HTML문서의 head태그 안에 있는 형태 (해당 HTML문서내에서만 적용)
  • Inline CSS - CSS소스가 HTML태그 바로 뒤에 있는 형태
    • Inline CSS - CSS소스가 HTML태그 바로 뒤에 있는 형태. 일관된 디자인 불가, 유지보수 어려움 (CSS적용 우선순위가 가장 높음)

선택자 종류 중 대표 3가지

  • tag선택자() - tag 그대로 불러와서 선택자로 적용. (강제적, 자동적 적용)
  • ID선택자 (#) - tag에 ID명을 지정하고 ID값을 불러와서 선택자로 적용.구조적인 용도
    • (ID값은 중복적용이 불가하므로 같은 스타일을 다른 요소에 적용할 수 없음)
  • class선택자 (.) - tag에 class명을 지정하고 class값을 불러와서 선택자로 적용. 디자인적인 용도
    • (class값은 중복적용이 가능하므로 같은 스타일을 여러 요소에 동시 적용할 수 있음)
